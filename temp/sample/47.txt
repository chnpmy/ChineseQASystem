2012年10月11日13时，[[瑞典文学院]][[诺贝尔奖评审委员会]]、瑞典文学院常任秘书[[彼得·恩隆德]]宣布，本年度[[诺贝尔文学奖]]的获得者是中国作家[[莫言]]，彼得·恩隆德说“（莫言）以[[幻觉现实主义]]融合了[[民间故事]]、[[历史]]与[[当代]]。”<ref>{{cite web |url=http://article.yeeyan.org/view/245405/324909 |title=莫言到底是不是“魔幻现实主义融合了民间故事、历史与当代社会” |accessdate= 2012年10月14日|publisher=译言网 |work= |language =zh-cn}}</ref>（{{lang-en|who with hallucinatory realism merges folk tales, history and the contemporary}}）<ref>{{cite web |url=http://www.nobelprize.org/nobel_prizes/literature/laureates/2012/press.html# |title=The Nobel Prize in Literature 2012 Mo Yan |accessdate=2012 |publisher=诺贝尔奖官方网 |work= |language =zh-cn}}</ref> 莫言成为第一位获得诺贝尔文学奖的[[中华人民共和国国籍|中国籍]]作家。

== 提名 ==
瑞典文学院常任秘书彼得·恩隆德曾透露，2012年诺贝尔文学奖的提名一共有210位作家，其中46位是首次提名。<ref name="fish " >{{cite web |url= http://www.impactchina.com.cn/wenxueyishu/suibi/jianxun/2012-10-18/12554.html|title=2012年诺贝尔文学奖提名结束 |accessdate= 2012-10-18  |publisher=影响力中国 |work= |language =zh-cn}}</ref><ref name="tiger " >{{cite web |url=http://news.163.com/12/0305/07/7RQLH0JV00014AED.html |title=2012诺贝尔文学奖提名结束 |accessdate=2012-03-05  |publisher=网易 |work= |language =zh-cn}}</ref> 欧美博彩公司和美国《纽约时报》刊文说，中国作家莫言、曹乃谦、[[张一一]]获得提名。<ref name="bull " >{{cite web |url=http://news.jschina.com.cn/system/2012/04/26/013225427.shtml |title=2012诺贝尔文学奖提名进入复评 中国2作家入围 |accessdate=2012-04-26  |publisher=中国江苏网 |work= |language =zh-cn}}</ref>

2012年4月，瑞典文学院诺贝尔奖评审委员会否决了210人中的190人，其中20人进入第二轮评选。<ref name="fish " /><ref name="tiger " /> 中国的莫言、[[曹乃谦]]<ref>{{cite web |url=http://news.ifeng.com/mainland/detail_2012_04/26/14171541_0.shtml |title=山西作家曹乃谦入2012年诺贝尔文学奖复评名单 |accessdate=2012年4月26日 |publisher=凤凰网 |work= |language =zh-cn}}</ref>进入20强的名单。<ref name="bull " />

瑞典文学院诺贝尔奖评审委员会在第二轮评选中否决15人，留下5人进行秘密投票，由票数决定最后得主。<ref name="fish " /><ref name="tiger " /> 

== 获奖 ==
2012年10月11日13时，瑞典文学院诺贝尔奖评审委员会、瑞典文学院常任秘书彼得·恩隆德公布年度诺贝尔文学奖的得主是中国作家莫言，诺贝尔委员会新闻稿说：{{cquote| （莫言）以幻觉现实主义融合了民间故事、历史与当代。<ref>{{cite web |url=http://article.yeeyan.org/view/245405/324909 |title=莫言到底是不是“魔幻现实主义融合了民间故事、历史与当代社会” |accessdate= 2012年10月14日|publisher=译言网 |work= |language =zh-cn}}</ref>}}<br />{{cquote|who with hallucinatory realism merges folk tales, history and the contemporary.}}

== 官方反应 ==
2012年10月11日，[[中共中央政治局常委]][[李长春]]致信中国作家协会，贺信说：{{quote|随着我国[[改革开放]]和[[现代化]]建设的迅猛发展，[[中国文学]]迸发出巨大的创造活力，广大中国作家植根于人民生活和民族传统的深厚土壤，创作出一大批具有中国特色、中国风格、中国气派的优秀作品。莫言就是其中的杰出代表。莫言获得诺贝尔文学奖，既是中国文学繁荣进步的体现，也是我国综合国力和国际影响力不断提升的体现。他希望广大作家坚持以人民为中心的创作导向，贴近实际、贴近生活、贴近群众，创作出更多无愧于历史、无愧于时代、无愧于人民的优秀作品，为[[中华文化]]繁荣发展，为人类文明进步作出新的更大贡献。<ref>{{cite web |url=http://culture.ifeng.com/huodong/special/2012nuobeierwenxuejiang/content-3/detail_2012_10/12/18212883_0.shtml |title=李长春致信中国作协祝贺莫言获诺贝尔文学奖 |accessdate=2012年10月12日  |publisher=凤凰网 |work= |language =zh-cn}}</ref>}}

2012年10月11日，[[中国作家协会]]发表最莫言获奖的贺词：{{quote|欣闻莫言先生荣获2012年诺贝尔文学奖，我们表示热烈祝贺!<br />在几十年文学创作道路上，莫言对祖国怀有真挚情感，与人民大众保持紧密联系，潜心于艺术创新，取得了卓越成就。自上世纪八十年代以来，莫言一直身处中国文学探索和创造的前沿，作品深深扎根于乡土，从生活中汲取艺术灵感，从[[中华民族]]百年来的命运和奋斗中汲取思想力量，以奔放独特的民族风格，有力地拓展了中国文学的想象空间、思想深度和艺术境界。莫言的作品深受国内外广大读者喜爱，在中国当代文学史上占有重要地位。<br />莫言的获奖，表明国际文坛对中国当代文学及作家的深切关注，表明中国文学所具有的世界意义。希望中国作家继续勤奋笔耕，奉献更多精品力作，为人类的文化发展作出新的贡献！<ref>{{cite web |url=http://www.people.com.cn/24hour/n/2012/1012/c25408-19236223.html |title= 中国作协发表贺辞祝贺莫言获奖|accessdate= 2012年10月12日|publisher=人民网 |work= |language =zh-cn}}</ref>}}

值得一提的是，对于莫言获奖，[[中国共产党|中共]]当局政府同当年刘晓波获得[[2010年诺贝尔和平奖]]的态度截然相反。两年前，[[新华网]]就刘晓波获[[诺贝尔和平奖]]事件称：“刘晓波是什么人？他何德何能受到诺贝尔委员会的青睐<ref>[http://news.xinhuanet.com/world/2010-10/26/c_12703288.htm 刘晓波其人其事|date=2010年10月26日]</ref>。”而对于莫言获奖，则进行了全方位的报的，其中不乏溢美褒奖之词<ref>[http://news.xinhuanet.com/world/2012-10/11/c_113343763.htm 中国作家莫言获2012年诺贝尔文学奖|date=2012年10月11日]</ref>。

== 评价 ==

2012年10月11日，诺贝尔文学奖委员会宣布2012年诺贝尔文学奖获得者为莫言。此事不仅在文学界产生影响，也引起社会各界广泛讨论。中华人民共和国官方和大部分中文作家对莫言表示祝贺和肯定，一些反共作家和政治人士反对诺贝尔奖委员会的决定，质疑莫言获奖。

=== 正面 ===
[[中国大陆]]和[[台湾]]的中文文学界的[[王蒙]]、[[陈忠实]]、[[余华]]、[[龙应台]]、[[陈丹青]]、[[阎连科]]、[[苏童]]、[[迟子建]]、[[李锐 (小说家)|李锐]]、[[周国平]]等人在莫言获奖时大都表达出对其作品和写作态度的肯定<ref>[http://udn.com/NEWS/WORLD/WORS1/7424963.shtml 黃春明 曾送他隨身用的鋼筆]</ref><ref>{{cite web | url=http://news.163.com/12/1012/00/8DIVN61R00014JB5.html | title=湖北文联主席：中国文学要以中国风格走向世界 | publisher=新华网 | accessdate=10 13, 2012}}</ref><ref>{{cite web | url=http://culture.ifeng.com/huodong/special/2012nuobeierwenxuejiang/content-1/detail_2012_10/12/18206231_0.shtml | title=龙应台：望莫言获奖能令“中国心灵打开” | publisher=凤凰网 | accessdate=10 13, 2012}}</ref><ref>{{cite web | url=http://culture.ifeng.com/huodong/special/2012nuobeierwenxuejiang/content-1/detail_2012_10/12/18207890_0.shtml?_from_ralated | title=朱天心：莫言在大陆排名前几位 符合西方对东方的想象 | publisher=凤凰网 | accessdate=10 13, 2012}}</ref>。

莫言作品的译著人葛浩文、[[陈安娜]]等人对其获奖表示肯定，而中国媒体以及[[汉学家]][[沃尔夫冈·顾彬|顾彬]]认为莫言获奖源自于他们卓越的翻译版本有着强大的全球推广作用，<ref>{{cite web |url=http://book.sina.com.cn/cul/c/2012-10-12/1111345045.shtml |title=德国汉学家顾彬：莫言讲的是荒诞离奇的故事 |accessdate=2012年10月12日  |publisher=新浪网 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://sh.sina.com.cn/citylink/jk/t_xa/2012-10-12/1348119274.html |title=莫言获奖离不开翻译陈安娜 |accessdate=2012年10月12日 |publisher=新浪网 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://pinglun.eastday.com/p/20121013/u1a6917988.html |title= 解放日报:莫言得奖,翻译有功|accessdate=2012年10月13日 |publisher=东方网 |work= |language =zh-cn}}</ref>莫言著作的英文译者[[葛浩文]]（Howard Goldblatt）认为莫言的经历和中国很多作家一样。他认为莫言习惯于远离政治，但是尊敬那些中国的异见人士。2000年[[高行健]]获得诺贝尔文学奖时，莫言是极少数站出来为之说话的人。<ref>{{cite news|author=John Ruwitch|title=China Nobel winner Mo likely to steer clear of politics: translator|url=http://www.reuters.com/article/2012/10/12/us-nobel-moyan-translator-idUSBRE89B06220121012|accessdate=2012-10-16|newspaper=reuters|date=Oct 12, 2012}}</ref>

==== 回应 ====

诺贝尔奖评审委员会否定金钱利益考虑以及反驳了关于莫言政治立场谴责的批评，评委马悦然认为，人们不能简单地把中国人分为“当局的忠诚者”和“异议人士”<ref>{{cite web |url=http://www.dw.de/%E8%AF%BA%E5%A5%96%E5%A7%94%E5%91%98%E4%BC%9A%E4%B8%BA%E6%8E%88%E5%A5%96%E8%8E%AB%E8%A8%80%E8%BE%A9%E6%8A%A4/a-16313419|title=诺奖委员会为授奖莫言辩护|accessdate= 2012-10-24|publisher=德国之声|work= |language =zh-cn}}</ref>。

10月12日，莫言举行获诺贝尔文学奖后的新闻发布会，会上，莫言回应并无迁居海外的意向。他表示不会后悔曾抄写毛泽东在延安文艺座谈会上的讲话。他表示该“讲话虽然是有着巨大的局限性，过分强调文学与阶级的关系而忽略了文学的人性，但也觉得其中有值得认可的部分，并且在当时那种社会历史的背景下，对于推翻腐朽的政权产生了积极的作用。”<ref>{{cite web |url=http://culture.ifeng.com/huodong/special/2012nuobeierwenxuejiang/content-3/detail_2012_10/12/18218915_0.shtml?_from_ralated |title=莫言：抄写《延安文艺讲话》不后悔 它有合理成分 |accessdate= 2012年10月12日|publisher=凤凰网 |work= |language =zh-cn}}</ref>  他认为作为80年代的作家之一，开始写作时已经意识到延安讲话的精神有局限性，已经无法满足文学创作的需要，他的文学创作都是在突破这种局限。莫言认为他的写作并“不限于党派，作家不为党派服务”，而是“站在人的立场，写人的情感、人的命运”，他的小说是“大于政治的”；媒体以他出国与否去判断其是否爱国，是非常片面的事情。<ref>{{cite web | url=http://news.qq.com/a/20121013/000618.htm?pgv_ref=aio2012&ptlang=2052 | title=莫言：不会离开这个国家 不后悔抄写延安讲话。 | publisher=东方网 | date=2012年10月13日08:54 | accessdate=10 13, 2012}}</ref><ref>{{cite web |url= http://culture.ifeng.com/huodong/special/2012nuobeierwenxuejiang/content-3/detail_2012_10/12/18219386_0.shtml|title=莫言：文学奖不是政治奖 作家不为党派服务(全文) |accessdate= 2012年10月12日 |publisher=凤凰网 |work= |language =zh-cn}}</ref>

当日新闻发布会上，莫言回答关于[[刘晓波]]的提问：「我对他后来的很多的活动都不太了解。但是我现在希望­他能够尽早地获得自由，尽早地能够健康地获得他的自由，然后我觉得他完全可以研究他的政治，研究他的社会体制。」<ref>{{cite news | url=http://news.sina.com.tw/article/20121012/8043373.html | title=諾貝爾文學獎得主莫言　希望劉曉波獲釋 | publisher=新浪新聞 | date=2012-10-12 08:00 | accessdate=10 14, 2012}}</ref><ref>[http://www.youtube.com/watch?v=lCo_HouZd2o 莫言：希望刘晓波尽快获得自由 ]</ref>
而在莫言领奖的会上，自由亚洲电台记者要求莫言重复之前“声援刘晓波”的言论，莫言回答：“大家的时间都宝贵，没有必要重复。我当天回答的内容你明明都知道了，你为什么要我重复呢？大家时间都宝贵需要均摊，我希望你不要揪住一个问题不放”。<ref name="nybj"/>

对于课文的批评，《[[人民文学]]》主编[[李敬泽]]、《[[收获]]》主编[[叶开]]等提醒政府当局不要低估学生的理解能力、不要对莫言文字做风马牛不相及的解读。<ref>{{cite web |url=http://culture.ifeng.com/huodong/special/2012nuobeierwenxuejiang/content-1/detail_2012_10/15/18259792_0.shtml |title=叶开担忧莫言作品进语文教材：一经解读就很可怕 |accessdate=2012年10月15日 |publisher=凤凰网 |work= |language =zh-cn}}</ref>

== 莫言热 ==
莫言获奖引发了中文界和当地政府的亢奋，提出了“文化搭台，经济唱戏”的观念：莫言的家乡[[高密市人民政府]]投资6.7亿打造“红高粱文化休闲区”，将用千万元种植万亩[[红高粱]]，而老家管委会主任范珲表示先扔一千万，赔本也要种；莫言文学馆也将重新建造更大的；[[山东省]]的旅游口号将修改为“一山一水一圣人一文豪”；莫言家的[[萝卜]]、[[土豆]]被视为“神物”；莫言作品即将列入高中语文教材；影视公司将拍摄莫言作品；莫言老家要重新建筑，辖区官员直接对莫言父亲说儿子不是他的儿子，房子不是他的房子了<ref>{{cite web |url=http://roll.sohu.com/20121019/n355199845.shtml |title= 儿子已经不是你的儿子屋子也不是你的屋子了|accessdate=2012年10月19日 |publisher=搜狐网 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://hb.people.com.cn/n/2012/1018/c194083-17597171-1.html |title=莫言诺奖后 “儿子已经不是你的儿子了” |accessdate=2012年10月18日 |publisher=人民网 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://culture.ifeng.com/huodong/special/2012nuobeierwenxuejiang/content-3/detail_2012_10/19/18372584_0.shtml |title=评“莫言热”：高密难道要改称“红高粱市”？ |accessdate=2012年10月19日 |publisher=凤凰网 |work= |language =zh-cn}}</ref><ref>{{cite web |url=http://culture.ifeng.com/huodong/special/2012nuobeierwenxuejiang/content-1/detail_2012_10/18/18349910_0.shtml |title= 评：死了“莫言家乡打造万亩红高粱”这条心|accessdate= 2012年10月18日|publisher= 凤凰网|work= |language =zh-cn}}</ref>。

==领奖言论争议==
=== 审查制度 ===
[[美联社]]对莫言的报导则以莫言对审查制度的态度为重点，认为：被批评为身为中共党员、不愿反对中国政府、不维护言论自由并且支持中共背景的中国作家协会的莫言，认为审查制度和出入境安检是一样的<ref name="apst">{{cite news|url=http://www.guardian.co.uk/world/2012/dec/07/mo-yan-censorship-nobel |title=Censorship is a must, says China's Nobel winner | work=Associated Press in Stockholm guardian.co.uk, Friday 7 December 2012 03.04 GMT | date=2012-12-06 |accessdate=12 07, 2012}}</ref>。莫言自称“反感所有的检查”，他称：“我从来没有赞美过新闻审查这种制度，但我想新闻审查每个国家都存在。但是审查的尺度和标准都不一样。如果没有新闻审查，大家都可以在任意污蔑和诽谤人家” ，他又认为“新闻审查应该遵守的最高准则是，只要不违背实事真相的都不应该审查，违背了实事真相的都要审查”<ref name="nybj">{{cite news|url=http://cn.nytimes.com/article/culture-arts/2012/12/06/cc06moyanpress/ | title=莫言赴瑞典领奖避谈刘晓波 |work=《纽约时报》北京分社 报道 2012年12月6日 | date=2012-12-06 |accessdate=12 07, 2012}}</ref>，美联社则将其回答截断并翻译为“新闻审查应该有最高准则”。（“censorship should have the highest principle”），并以“中国诺奖获得者称审查是必须”作为标题<ref name="apst"/>被问到审查制度是否有利于写作时，莫言回答说：“如果一个作家认为他在完全自由的状态下能够写出最伟大的作品，我认为这是一种幻想。反过来说，如果一个作家在不自由或者不太自由的情况写不出好作品，也是假话。关键是作家自己的内心是否自由，关键是作家能否站在超越了政治和阶级的立场来写作，包括作家的爱心，包括那些在背后咬牙切齿憎恨你的人，也要把他们当人看，给予他们深深的同情”<ref name="nybj"/>。

《大纪元时报》和台湾中央通信社将[[美聯社]]的報導解读为：莫言是[[中國作家協會]]副主席，因為沒有更大膽的發聲捍衛言論自由，原已遭致批評。但莫言2012年12月6日在[[瑞典]]領獎前夕的公開發言更惹議，他表示，（[[中共]]）當局的審查制度，像機場安檢一樣有必要。「不覺得審查制度應阻擋真相，但是任何誹謗或謠言『應該被封殺』。」「但是我也希望言論審查制度，應該有更高原則。」」2009年[[諾貝爾文學獎]]得主[[赫塔·米勒]]則批評，評審團選擇莫言是一個『大災難』，並指責莫言為中共審查制度辯護。<ref>{{cite news | url=http://www.cna.com.tw/News/aCN/201212070177-1.aspx | title=莫言：言論審查如安檢 | work=(台灣)中央通訊社 | date=2012-12-06 | accessdate=12 07, 2012 | author=綜合外電}}</ref><ref>{{cite news | url=http://www.epochtimes.com/b5/12/12/7/n3746914.htm | title=前諾獎得主：評審團選擇莫言是「大災難」 | work=[[大紀元時報]]--編譯自《[[美聯社]]》 | date=2012年12月7日 | accessdate=12 07, 2012 | author=秦雨霏編譯}}</ref>

'''戴尔·莫滕森'''（{{lang-en|'''Dale Thomas Mortensen'''}}，{{bd|1939年|2月2日|2014年|1月9日|catIdx=Mortensen}}），[[美国]][[经济学家]]，前[[西北大学]]教授，以对[[劳动经济学]]尤其是[[失業#摩擦性失業|摩擦性失业]]理论的研究而知名。2010年12月10日，与[[彼得·戴蒙德]]、[[克里斯托弗·皮萨里德斯]]一同获得[[诺贝尔经济学奖]]。<ref>{{cite web|url=http://nobelprize.org/nobel_prizes/economics/laureates/2010/mortensen.html|title=Dale T. Mortensen - Biographical|date=2010-10-11|publisher=Nobelprize.org}}</ref>

== 经历 ==
莫滕森分别在[[威拉米特大学]]与[[卡内基梅隆大学]]获得学士与博士学位。1965年起，开始在[[西北大学]]任教，主要研究领域为[[劳动经济学]]。<ref name="qq">{{cite web|url=http://finance.qq.com/a/20101011/006809.htm|title=2010年诺贝尔经济学奖获奖者戴尔-莫滕森简历|date=2010-10-11|publisher=腾讯财经}}</ref>

1965年他曾获亚历山大·亨德森奖（Alexander Henderson Award）。1979年加入[[美国计量经济学会]]（Econometric Society）。2000年，成为[[美国文理科学院]]院士。2005年获IZA劳动经济学奖。2010年12月10日，获得[[诺贝尔经济学奖]]。<ref name="qq" />

2014年1月9日，莫滕森在[[美國]][[伊利諾州]][[庫克縣]][[威爾明特]]逝世，享年75岁。<ref>[http://hk.apple.nextmedia.com/international/art/20140111/18587925 經濟學諾獎得主莫滕森逝世]</ref>

== 背景 ==
1992年，高行健获得[[法国政府]]颁发的“[[藝術及文學勳章 (法國)|艺术与文学骑士勋章]]”。

== 评价 ==
=== 正面 ===
[[中国大陆]]、[[香港]]和[[台湾]]的中文文学界的[[余杰]]、[[王朔]]、[[舒乙]]、[[董桥]]、[[刘震云]]、[[何志云]]、[[刘再复]]、[[方梓勋]]、[[王吉隆]]、[[莫言]]、[[龙应台]]、[[南方朔]]等在高行健获奖时大都表露出对其作品和写作态度的肯定，指出其作品融合了[[西方]][[现代文学]]技巧和[[东方]]儒释道的精神内涵。<ref name="man " >{{cite web |url=http://www.zaobao.com/special/newspapers/2000/pages5/zb141000a.html |title= 高行健获颁诺贝尔文学奖反应；中国文学界有人斥为“荒唐”|accessdate= |publisher=联合早报网 |work= |archiveurl=http://www.zaobao.com/special/newspapers/2000/pages5/zb141000a.html  |archivedate=  2012年10月25日 |language =zh-cn}}</ref><ref>{{cite web |url=http://www.zaobao.com/special/newspapers/2000/pages5/zb141000b.html |title=高行健著作在台北被抢购一空 |accessdate=2000 |publisher=联合早报网 |work= |archiveurl=http://www.zaobao.com/special/newspapers/2000/pages5/zb141000b.html |archivedate= 2012年10月25日 |language =zh-cn}}</ref>

=== 争议 ===
高行健获奖得到中国部分人士的肯定却也招来中国官方及部分人士的批评。

[[成晓明]]、[[朱天文]]、[[张晓风]]、[[马森]]、[[夏志清]]等认为其小说作品数量少、不够优秀，“非常荒谬”、“太荒唐”。<ref name="man " />

[[中国外交部发言人]][[朱邦造]]：{{quote|诺贝尔文学奖评委会最近的行动再次表明，诺贝尔文学奖已经被用于别有用心的政治目的，不值一评。<ref name="fish " >{{cite web |url= http://news.21cn.com/today/2006/09/11/2970908.shtml|title=2000年度诺贝尔文学奖备受中国官方批评 |accessdate=2006-09-11  |publisher=21CN |work= |archiveurl=http://news.21cn.com/today/2006/09/11/2970908.shtml |archivedate= 2012年10月25日  |language =zh-cn}}</ref>}}

[[中国作家协会]]：{{quote|中国有许多举世瞩目的优秀文学作品和文学家，诺贝尔文学奖评委会对此并不了解，看来，此举不是从文学角度评选，而是有其政治标准。<ref name="fish " /><ref>{{cite web |url=http://culture.ifeng.com/huodong/special/2012nuobeierwenxuejiang/content-4/detail_2012_10/11/18190867_0.shtml?_from_ralated |title=历史资料：2000年中国作协就高行健获诺贝尔奖的表态 |accessdate= 2012年10月11日 |publisher=凤凰网 |work= |archiveurl=http://culture.ifeng.com/huodong/special/2012nuobeierwenxuejiang/content-4/detail_2012_10/11/18190867_0.shtml?_from_ralated  |archivedate= 2012年10月25日 |language =zh-cn}}</ref>}}

中共官方报刊《[[人民日报]]》专文《将“诺贝尔文学奖”授予高行健严重伤害了中国人民的感情》：{{quote|瑞典文学院是一贯反动、一贯敌视红色政权、一贯敌视中国人民的彻头彻尾的反动组织。<ref>{{cite journal | language =zh-cn | author = | title =《将“诺贝尔文学奖”授予高行健严重伤害了中国人民的感情》| journal =《人民日报》 | volume = | year = 2010年 | pages = | issn = CN 11-0065}}</ref><ref>{{cite web |url=http://www.people.com.cn/GB/guandian/183/8135/8136/20001019/728165.html |title= 2000年诺贝尔文学奖违背了诺贝尔遗嘱|accessdate=2000年10月19日 |publisher=人民网 |work=  |archiveurl=http://www.people.com.cn/GB/guandian/183/8135/8136/20001019/728165.html |archivedate=2012年10月25日|language =zh-cn}}</ref>}}

== 媒体报道 ==

'''俄羅斯科學院'''（{{lang-ru|Росси́йская Акаде́мия Нау́к}}，俄文縮寫'''PAH'''）是[[俄羅斯]]的國家學術機構。是一個公共且非營利組織，也是世界上重要研究機構之一。1724年由[[彼得一世]]建立，称彼得堡科学院，十月革命后改用苏联科学院，1934年迁至現今的總部，位於[[莫斯科]]。蘇聯解體後改為現名至今。
